# ReLiveTV

A nostalgic web app that broadcasts curated YouTube content as scheduled TV channels.
Watch on your laptop, control with your phone — no pause, no rewind, just channels.

## Concept

- **5 channels**: Comedy, Crime, News, Music, Sports
- **No pause, no rewind, no fast-forward** — just channel up/down
- **Synchronized broadcast** — everyone tuned to the same channel sees the same moment
- **Two-screen experience** — desktop is the TV, phone is the remote




## Tech Stack

| Tier | Tech |
|------|------|
| Frontend | React 18, Vite, Tailwind CSS, Socket.IO client, Supabase JS SDK |
| Backend | Node.js 20+, Express, Socket.IO, Prisma, Zod |
| Database | PostgreSQL (Supabase) |
| Auth | Supabase Auth (JWT) |
| Hosting | Netlify (FE), Render (BE), Supabase (DB) |
